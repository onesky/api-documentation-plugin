# Order Resources

    GET orders/:order_id

## Description
Return details of an order


## Requires authentication
- Details described [here](/README.md#authentication)


## Parameters
NONE


## Example
**Request**

    GET https://api.plugin.onesky.io/1/projects/:project_id/orders/:order_id
