# Account Resources

    POST sign-up

## Description
Register account in [onesky](http://oneskyapp.com)


## Requires authentication
NONE


## Parameters
- `email` _(required)_ - User email to register account.


## Example
**Request**

    POST https://api.plugin.onesky.io/1/accounts/sign-up

**Response**
```
status 201 Created
```
